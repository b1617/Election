# Election

- Simple decentralized voting application based on smart contract. 

## Installation and Run

- Install Ganache with Truffle

- Use the [npm](https://www.npmjs.com/) to install.

```bash
npm install
```
```bash
npm start
```

Open browser
```
http://localhost:3000
```

## Technologies

- Ganache
- Truffle
- Solidity
- React
- Mocha
